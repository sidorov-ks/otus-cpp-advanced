Detailed Description

Структура, реализующая обёртку над std::ostream. Форматирует выводимые объекты как IP-адреса (см. также перегрузки метода operator<<)

ip_proxy& ip_proxy::operator<< ( T const &  val)
inline

Выводит IP-адрес, заданный в виде числа.

Данная реализация выводит каждый из четырёх октетов из младших 4 байтов числа, выделяя на каждый октет по 1 байту, начиная с младших октетов. Так, для val = 2130706433 будет выведен IP-адрес 127.0.0.1, поскольку val == (1 + (127 << 24)).
